# Assets Directory

Place your app icons here:

- `icon.png` - App icon (1024x1024px)
- `splash.png` - Splash screen (1242x2436px)
- `adaptive-icon.png` - Android adaptive icon (1024x1024px)
- `favicon.png` - Favicon (48x48px)

You can use any image editing tool to create these icons.

For now, you can use placeholder images or the default Expo icons will be used.
